Во времена Руси древней 
Люд забавлялся добротной песней. 
Была далече тоска да скука, 
Средь деревень играла эта музыка...

Драм. Драм. Драм. Драм.

Драм для Ивана, 
Драм для Михаила, 
Драм для Георгия, 
Драм для Киррила. 
Драм для Ольги, 
Драм для Ирины, 
Драм для Светланы, 
Драм для Екатерины. 
Драм. 

Славься Русь и ее корни, 
Драм-н-бейс лихой, задорный. 
Ух, не даром эти звуки 
В пляс лихой пускают наши ноги-руки. 

Драм. Драм. Драм. Драм.

Драм для Павла, 
Драм для Ярослава, 
Драм для Александра, 
Драм для Вячеслава. 
Драм для Натальи, 
Драм для Евгении, 
Драм для Татьяны, 
Драм для Марии. 

Драм. 
Эх, лихой да драм.
